# comtrans
Сервис управления парком частного и общественного транспорта

## Стек технологий

- База данных: PostgreSQL
- Фронтенд: React + Redux, MUI
- Бэкенд: .NET Core 6

## Структура проекта

- **client** - исходный код web- и mobile-клиентов
- **services** - исходный код сервисов бэкенда (.NET)
- **db** - БД
